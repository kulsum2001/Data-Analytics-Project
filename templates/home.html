Discover the sentiment behind Amazon Alexa reviews with advanced machine learning.

This project leverages machine learning to analyze customer reviews of Amazon Alexa products, classifying sentiments as positive or negative. Using a dataset of 3,150 reviews, a neural network model trained on TF-IDF features provides detailed insights into customer feedback. Explore comprehensive visualizations, performance metrics, and downloadable reports to understand user sentiments and model accuracy with ease.

About This Project

This project analyzes customer reviews of Amazon Alexa products using a neural network to classify sentiments as positive or negative. The dataset contains 3150 reviews, and the model is trained using TF-IDF features. Click "Get Started" to explore the results!
